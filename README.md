## 📦 Project Overview

Catshop is a simulated retail system with multiple interfaces:

- Customer Client  
- Cashier Client  
- BackDoor Client  
- Pick Client  
- Display Client  
- Collect Client  

Customers can:
- Browse items by product ID
- View quantity and price
- Place or remove orders
- Receive unique order numbers
- Collect items when ready
